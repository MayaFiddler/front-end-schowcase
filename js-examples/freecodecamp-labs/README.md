# FreeCodeCamp JavaScript Labs

This folder contains selected solutions to JavaScript labs from
the FreeCodeCamp curriculum.

The examples demonstrate:
- loops
- arrays and objects
- problem solving
- algorithmic thinking

The code has been reviewed and slightly refactored for readability
and consistency with real-world coding practices.

## Files

### js01-sum-array.js
Sums values in an array using iteration and basic logic.

### js02-object-iterations.js
Iterates over object properties and values.
Demonstrates working with JavaScript objects and loops.
