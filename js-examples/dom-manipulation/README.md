# DOM Manipulation Examples

This folder contains JavaScript examples focused on interacting with the DOM.

## Covered concepts
- querySelector / querySelectorAll
- event listeners
- classList manipulation
- basic UI state handling
- form interaction and validation feedback

## Files

### toggle-menu.js
A simple script that toggles a mobile navigation menu.
Demonstrates event handling and class toggling.

### interactive-form.js
Handles basic frontend form interactions and validation feedback.
Focuses on user experience and visual response without backend integration.
