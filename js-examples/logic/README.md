# JavaScript Logic Examples

This folder contains small JavaScript files focused on pure logic and calculations,
without DOM interaction.

## Covered concepts
- functions
- conditional logic
- basic arithmetic
- clean and readable code structure

## Files

### calculator.js
A basic calculator logic implementation demonstrating functions and conditionals.

### tip-calculator.js
Calculates tips based on bill amount and percentage.
Shows real-world use of variables, functions, and simple math logic.
