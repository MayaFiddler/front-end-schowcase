# SpeakClear - EdTech Landing Page

A responsive landing page concept for an AI-powered language learning application. I built this project to demonstrate my ability to create clean, accessible user interfaces using HTML, CSS, and Vanilla JavaScript.

### 🧠 The "Why" Behind the Design
With my background in linguistics and education, I know that language learners often feel anxious when practicing pronunciation. I designed this page with a focus on **low cognitive load**:
- Lots of whitespace to prevent overwhelm.
- A calming color palette.
- Clear, readable typography.

### 🛠️ Technical Highlights
- **Semantic HTML5** for accessibility.
- **CSS Flexbox/Grid** for a fully mobile-responsive layout.
- **Vanilla JavaScript** to power the interactive "Mock Audio Recording" button, demonstrating DOM manipulation and event handling.
- **Accessibility:** Included a Dark/Light theme toggle to accommodate user visual preferences.
